在工程中综合排序：

  • 先判断是否为基础类型
  • 如果是基础类型则使用快排，分治时小于60则插排（不用保持稳定性）
  • 如果是自定义数据类型归并排序，分治时小于60则插排（保持稳定性）
  • 如果数据长度很短则使用插排,插排常数项极低，小样本情况很快

对数器

写在前面：通常为了验证我们所写排序算法的正确性需要使用到对数器。

对数器的概念和使用:

  • 有一个需要测试的方法a
  • 实现一个绝对正确但是复杂度不好的方法b
  • 实现一个随机样本产生器
  • 实现比对的方法
  • 把方法a和方法b比对很多次来验证方法a是否正确
  • 如果有一个样本出错，打印分析之
  • 当样本足够多依然比对正确，那么方法a是正确的

下面是关于对数器的Java版本代码

\\生成随机数数组，设定随机数据大小，以及随机数据最大值
public static int[] generateRandomArray(int maxSize, int maxValue) {
    int[] arr = new int[(int) ((maxSize + 1) * Math.random())];
    for (int i = 0; i < arr.length; i++) {
        arr[i] = (int) ((maxValue + 1) * Math.random()) - (int) (maxValue * Math.random());
    }
    return arr;
}
\\拷贝操作，两个比较器对同一数据进行排序
public static int[] copyArray(int[] arr) {
    if (arr == null) {
        return null;
    }
    int[] res = new int[arr.length];
    for (int i = 0; i < arr.length; i++) {
        res[i] = arr[i];
    }
    return res;
}
\\之后就是不断的循环比较(比较大量的次数如果没有问题，那么应该是没有问题的)
for (int i = 0; i < testTime; i++) {
    int[] arr1 = generateRandomArray(maxSize, maxValue);
    int[] arr2 = copyArray(arr1);
    if (maxGap(arr1) != comparator(arr2)) {
        break;
    }
}

堆排序

堆排序（HeapSort）通常是利用树这种数据结构所设计的一种排序算法，其按照完全二叉树来设计的。
完全二叉树： 除了最后一层之外的其他每一层都被完全填充，并且所有结点都保持向左对齐。
因为排序通常是对数组进行排序，那么根据完全二叉树的特点，我们可以知道：

  • 父节点i的左子节点为2*i+1
  • 父节点i的右子节点为2*i+2
  • 子节点的父节点在(i-1)/2(向下取整)

堆又分为大根堆和小根堆。大根堆即对于任意一个子节点，其数值均小于父节点的值。小根堆则相反。

之后就是建堆的操作，以大根堆为例。

堆的插入

其步骤为：每次插入节点的时候，不断的向上查找节点，如果比父节点大则交换之，直到遇到小于等于父节点。 复杂度为O（logn）
具体代码实现如下: 

// 堆的插入，往上查找直到小于等于父节点，index为插入节点的位置
public static void heapInsert(int[] arr, int index) {
    while (arr[index] > arr[(index - 1) / 2]) {
        swap(arr, index, (index - 1) / 2);
        index = (index - 1) / 2;
    }

堆的调整

每当我们取出堆顶的元素时候，需要对堆进行调整以保证堆保持在最大堆状态。
其步骤为：先将堆顶与尾部交换，然后此时头部较小那么需要调整，先从左右子节点中挑一个大的，然后不断向下寻找直到不比子节点小了就停止。调整代价为O（logn），具体代码为：

public static void heapify(int[] arr, int index, int size) {
    int left = index * 2 + 1;
    while (left < size) {
        int largest = left + 1 < size && arr[left + 1] > arr[left] ? left + 1 : left; //找到左右最大值
        largest = arr[largest] > arr[index] ? largest : index; //比较
        if (largest == index) {
            break;
        }
        swap(arr, largest, index); //与最大的交换，然后
        index = largest;
        left = index * 2 + 1;
    }
}

实际使用

实际使用中我们通常采用优先队列的方式设置大根堆和小根堆。Java中为PriorityQueue，可以使用自定义比较器的方式设置比较的数据结构。

常见问题：

  • 一个流不断吐出数，随时找到吐出流中的中位数，吐出的数是无序的。求吐出所有数的中位数。
  • 经典top k问题，给定大量数据找出top100位数。
  • 剑指offer 最小的k位数

快速排序

快速排序指的是在数组中选择一个划分点，然后将数据划分为两个部分：一个比划分点小的部分，一个比划分点大的部分。然后不断迭代重复这个过程在左右两边。最后直到排序完毕。
具体代码如下: 

public static void quickSort(int[] arr){
    if(arr == null || arr.length < 2){
        return;
    }
    quickSort(arr,0,arr.length-1);
}
public static void quickSort(int[] arr, int l, int r){
    if(l < r){
        swap(arr, l+(int)(Math.random()*(r-l+1)), r);//随机选取一个数放在最后作为快排目标
        int[] p = partition(arr, l, r);//一遍快排给出边界
        quickSort(arr, l, p[0]-1);//左边快排
        quickSort(arr,p[1]+1,r);//右边快排
    }
}
public static int[] partition(int[] arr, int l, int r){
    int less = l-1;
    int more = r;
    while(l < more){
        if(arr[l] < arr[r]){
            swap(arr, ++less, l++);
        }else if(arr[l] > arr[r]){
            swap(arr, --more, l);
        }else{
            l++;
        }
    }
    swap(arr, more, r);
    return new int[]{less+1, more};//返回等于num的左右边界
}

常见问题:

  • 给定一个数组arr，和一个数Num,请把小于等于Num的数放在数组的左边，大于num的数放在数组的右边
  • 荷兰国旗问题：给定一个数组arr，和一个数Num,请把小于数放左边，等于的放中间，大于的放右边

归并排序

时间复杂度O（N*LogN）,额外空间复杂度O（N），采用的是分治的思想。
先左侧部分排好序，右侧部分排好序，然后利用外排的方式整体排序。具体代码如下：

public static void mergeSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    mergeSort(arr, 0, arr.length - 1);
}

public static void mergeSort(int[] arr, int l, int r) {
    if (l == r) {
        return;
    }
    int mid = l + ((r - l) >> 1);
    mergeSort(arr, l, mid);
    mergeSort(arr, mid + 1, r);
    merge(arr, l, mid, r);
}

public static void merge(int[] arr, int l, int m, int r) {
    int[] help = new int[r - l + 1];
    int i = 0;
    int p1 = l;
    int p2 = m + 1;
    while (p1 <= m && p2 <= r) {
        help[i++] = arr[p1] < arr[p2] ? arr[p1++] : arr[p2++];
    }
    while (p1 <= m) {
        help[i++] = arr[p1++];
    }
    while (p2 <= r) {
        help[i++] = arr[p2++];
    }
    for (i = 0; i < help.length; i++) {
        arr[l + i] = help[i];
    }
}

常见问题：

  • 小和问题：在一个数组中，每一个数左边比当前数小的数累加起来，就叫做这个数组的小和。求一个数组的小和。
  • 逆序对问题：一个数组中，左边的数如果比右边的大，则这两个数构成逆序对，请打印所有逆序对

排序稳定性

排序稳定性指的是序列中相同的值在经过排序之后，相同的值仍然保持相对位置不变。
例如在数据[2,1,2,3,4,2]中，经过一趟排序之后第一个2是否还在第二个2之前，对所有相同值都是如此。

稳定性的意义：在对象间进行比较的时候，希望原始序列能保存下来。

排序稳定的算法：

  • 冒泡排序：大值往下沉，遇到相同值换成下一个相同值往下沉，即可实现。O（n^2）
  • 插入排序：小值往前插，如果遇到相同的不插即可，换位下一个相同值进行插值。O(n^2)
  • 归并排序：归并的时候，相同的值先拷贝左边的再拷贝右边的，那么即可保持稳定。O（nLogn）
  • 桶排序：根据数据状况设计一个容器，每个容器里转若干数，稳定。O（N）
